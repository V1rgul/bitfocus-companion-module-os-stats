## Working on this module

- This is a [Companion connection module](https://github.com/bitfocus/companion-module-base) (`@companion-module/base` v2) targeting Node 22.
- Use `corepack yarn` as the package manager (not npm).
- If you modified project files, run `corepack yarn dist` once after all changes are complete and immediately before finishing the task. Do not run it during intermediate testing.
